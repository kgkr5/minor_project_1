# 🔐 QR-Based 2-Factor Authentication System with OTP Verification

A secure web application that implements **Two-Factor Authentication (2FA)** using **QR Code scanning** and **One-Time Password (OTP) verification**. The system enhances user authentication by requiring both password validation and OTP verification before granting access.

## 🚀 Features

- 🔑 User Registration and Login
- 📱 QR Code Generation for 2FA Setup
- 🔒 OTP Verification using Authenticator Apps
- 📧 Email OTP Support
- 🛡️ Secure Session Management
- ⚡ Backend Token Validation
- 🎨 Responsive User Interface

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT
- **QR Code:** QRCode Library
- **OTP:** OTPAuth / Speakeasy
- **Email:** Nodemailer

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/kgkr5/minor_project_1.git
```

2. Navigate to the project

```bash
cd minor_project_1
```

3. Install dependencies

```bash
npm install
```

4. Start the backend

```bash
npm start
```

5. Open the frontend in your browser.

## 📸 Project Workflow

1. User Registration
2. Login with Username & Password
3. Generate QR Code
4. Scan QR Code using Google Authenticator
5. Enter OTP
6. Secure Login Successful
